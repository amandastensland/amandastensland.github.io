Visa & Göm Element

För att visa och gömma menyn så används metoderna .hide och .show.

HTML

Kod för att skapa menyn och länkar.

<aside>
        <h2>Navigation</h2>
        <a href="#">Hem</a>
        <a href="#">Gästbok</a>
        <a href="#">Information</a>
        <a href="#">Saker</a>
</aside>

Kod för ikonen som triggar menyn. I detta exempel används ikoner från Bootstrap. För att använda dessa ikoner måste en länk till Bootstraps CSS-fil läggas till i <head> -taggen. Till exempel <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/css/bootstrap.min.css">

<span class="trigger-menu">
    <i class="glyphicon glyphicon-th-list"></i>
    <i class="glyphicon glyphicon-remove"></i>
</span>

CSS

Kod för att skapa meny-layouten.

aside {
    background: #293f50;
    position:fixed;left:0;top:0;
    width:15%;
    height:100%;
    color: white;
    overflow: auto;
    display:none;
}

aside h2 { margin:0.5em; }
aside a {
    display:block;
    color:white;
    margin:1em;
    border-bottom:1px solid rgba(255, 255, 255, 0.3);
}

Kod för ikonen som triggar menyn.

.trigger-menu {
    display: inline-block;
    font-size: 26px;
    margin: 2px 30px 0 0;
    float: left;
}
