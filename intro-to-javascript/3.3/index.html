DOM-objekt och jQuery

Exempel 1: Ändra en egenskap för ett HTML-elementet

Den här boxen är röd istället för gul..

$("#example1").css("background", "red");

Exempel 2: Lägga till nytt innehåll till ett HTML-elementet

Klicka på boxen för att lägga till text i den.

$("#example2").click(function(){
    $(this).text("Hello World!");
});

Exempel 3: Ta bort ett HTML-element

Klicka på den gula boxen för att ta bort den.

$("#example3").click(function() {
    $(this).remove();
});

Exempel 4: Utvinna information från ett HTML-element

$("#example4").click(function() {
    $(this).addClass();
});

Exempel 5: Lägga till och ta bort ett klass-attribut från ett HTML-element

Lägg till och ta bort en border runt den gula boxen med hjälp av knapparna.

$("#addBorder").click(function() {
    $("#example5").addClass("border");
});

$("#removeBorder").click(function() {
    $("#example5").removeClass("border");
});

Exempel 6: Mushändelser

Håll muspekaren ovanför den gula boxen så ändrar den färg.

$("#example6").mouseover(function() {
    $(this).css("background", "red");
});

$("#example6").mouseout(function() {
    $(this).css("background", "yellow");
});

Exempel 7: Dokument och fönsterhändelser

När man ändrar storleken på webbläsaren så ändras värderna.

$("#width").text($(window).width());
$("#height").text($(window).height());
$(window).resize(function() {
    $("#width").text($(window).width());
    $("#height").text($(window).height());
});

Exempel 8: Formulärhändelser

Skrivfältets bakgrund ändrar färg när det fokuseras.

$("#example8").focus(function() {
    $(this).css("background-color", "#cccccc");
});

$("#example8").blur(function() {
    $(this).css("background-color", "#ffffff");
});

Exempel 9: Tangentbordshändelser

Flytta blocket till höger eller vänster med piltangenterna.

$(document).keydown(function(e) {
    if (e.keyCode == 37) $("#example9").animate({left: "-=5"}, 0);
    if (e.keyCode == 39) $("#example9").animate({left: "+=5"}, 0);
});



Exempel 10: Kombinerade jQuery händelser med hover och toggle

Håll muspekaren över den första boxen för att se ett meddelande. Klicka på den andra boxen för att se ett till meddelande.

$("#example10-button1").hover(
    function() { $("#example10-text1").toggle(50); },
    function(){ $("#example10-text1").toggle(50); }
);

$("#example10-button2").click(function() {
    $("#example10-text2").toggle(50);
});

Exempel 12: Stoppa en händelses normala beteende

Stoppar en knapps normala beteende när man klickar på den (submit, d.v.s uppdatera sidan).

$("#example12-button").click(function(e) {
    e.preventDefault();
});

Exempel 13: Ta bort en händelse

Lägger till en klickfunktion till den gula boxen med hjälp av egenskapen "bind". Funktionen tas sedan bort med "unbind" när man klickar på boxen.

$("#example13-checkbox").click(function() {
    $("#example13-div").html("Klicka!");
    $("#example13-div").click(function() {
        alert("Klickad!");
        $(this).unbind("click");
        $("#example13-checkbox").prop("checked", false);
        $(this).text("Oklickbar...");
    });
});
