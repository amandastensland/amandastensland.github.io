Web Storage API

Med HTML5 LocalStorage och SessionStorage kan data sparas lokalt i användarens webbläsare. LocalStorage och SessionStorage är ett mer säkert alternativ till cookies. I LocalStorage går det att spara större mängder av data, utan att det påverkar webbläsarens prestanda. Vid användning av LocalStorage så överförs inte informationen till servern. LocalStorage lagrar datan utan någon satt tidpunkt att försvinna. SessionStorage lagrar endast data under en session. Det lagrade datat försvinner när webbläsaren eller tabben stängs.
