Responsiv stilsättning

Eftersom antalet användare som använder mobiltelefoner och mindre enheter för att visa innehåll från internet, ökar kravet på att en webbsida ska kunna anpassa sig till även dom minsta skärmarna.

Gemensam stilmall med @media-regeln

@media är en CSS regel som är ett sätt att inkludera och utesluta kod i stilmallen beroende på vilken enhet eller fönsterstorlek som används. Då behövs det bara en stilmall för att anpassa webbsidan till olika layouter. Den här metoden är ett bättre alternativ för mobila enheter då mindre HTTP-föfrågningar behöver göras.

CSS Mediatyper

@media all
Används för alla typer av enheter.
@media screen
Används för datorskärmar.
@media print
Används för skrivare.
@media handheld
Används för enheter som är mobila eller har små skärmar.

Exempel

I exemplet kommer bakgrunden på webbsidan att vara blå om webbläsaren befinner sig i datorskärmsläge och röd om webbläsaren befinner sig i utskriftsläge. 


@media screen {
    body {
        background-color: blue;
    }
}

@media print {
    body {
        background-color: red;
    }
}
