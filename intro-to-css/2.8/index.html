Separata stilmallar för olika mediatyper

För att optimera visningen av en webbsida för användaren så kan så kallada mediatyper användas. Syftet med dessa är att specifisera ett visst medium för en viss stilsättning av webbsidan och på så sätt göra det möjligt att justera utseendet på webbsidan beroende på vilken enhet som används för att få bästa möjliga användarupplevelse.

Den här webbsidan har konstruerats för tre olika mediatyper.

HTML Mediatyper

media="screen"
Webbläsare har vanligen screen som standardläge för att visa webbsidor och behöver egentligen inte specifiseras när man länkar till en stilmall.
media="print"
Mediatypen print används vanligen av webbläsaren om användaren vill skriva ut en webbsida. Webbläsaren kan då använda en mer utskriftsvänlig layout om en sådan finns specifiserad.
media="handheld"
Enheter som applicerar handheld-läget har ofta väldigt små skärmar som oftast skiftar mellan i porträtt- eller landskapsläge. Mediatypen kan specifiseras om det finns en klar tanke hur webbsidan bör se ut på enheter med mindre skärmar.

Exempel

I exemplet finns tre olika länkar till olika stilmallar. Beroende på vilken enhet som används så kommer den specifika stilmallen för den mediatypen att användas.


<head>
    <link href="2.8_style-screen.css" rel="stylesheet" type="text/css" media="screen">
    <link href="2.8_style-print.css" rel="stylesheet" type="text/css" media="print">
    <link href="2.8_style-handheld.css" rel="stylesheet" type="text/css" media="handheld">
</head>

CSS-konstruktioner för skrivare

För skrivare finns speciella CSS-konstruktioner som kan vara användbara när man designar layouter som ska vara utskriftsvänliga.

@page-regeln

@page
Är en regel som appliceras på utskriftslayouter. Regeln är inte implementerad i Firefox, men stöds av de senaste versionerna av Chrome, Internet Explorer, Opera och Safari. 

@page {
    margin: 0.5cm;
}

Sidbaserade egenskaper

page-break-after
Definierar om en sidbrytning ska ske efter ett specifikt element. 

@media print {
    footer { page-break-after: always; }
}
page-break-before
Definierar om en sidbrytning ska ske före ett specifikt element. 

@media print {
    h1 { page-break-before: always; }
}
page-break-inside
Definierar om en sidbrytning kan ske inom ett specifikt element. 

@media print {
    p { page-break-inside: avoid; }
}
size
Används för att definiera en storlek som webbsidan bör skrivas ut på för att uppnå bästa resultat. 
@page { 
 size: A4 landscape; 
}
orphans
Om en paragraf bryts till en ny sida så måste texten innan brytningen vara minst tre rader. 

@media print {
    p { orphans: 3; }
}
widows
Om en paragraf bryts till en ny utskriftssida så måste texten efter brytningen vara minst tre rader. 

@media print {
    p { widow: 3; }
}
marks
Lägger till beskärnings- och kryssmärken där dokumentet borde kapas. Kryssmärken används för att justera placeringen av utskriftens innehåll på pappret. 

@page {
    marks: crop cross;
}
