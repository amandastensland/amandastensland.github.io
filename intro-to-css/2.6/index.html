Tabeller och formulär

Tabellers utseende kan ändras och förbättras genom att använda CSS.

Table Layout

Table-layout egenskapen definierar layout-algoritmen som ska användas för tabellen.

Värde Beskrivning
auto Automatisk tabell-layout algoritm. Används som standard.
fixed Går snabbare att ladda in tabellen med värdet fixed. Den horisontella layouten beror bara på tabellens bredd, inte innehållet i tabellen.
initial Sätter värdet på egenskapen till dess standardvärde.
inherit Ärver egenskapen från ett förälder-element.

Exempel

table {
    table-layout:fixed;
}

Border Collapse

CSS-egenskapen border-collapse kan göra så att ramen, som i HTML-standard är dubbel, kan sammanfogas till endast en ram.

Värde Beskrivning
separate Ramar är åtskilda från varandra. Används som standard.
collapse Ramarna sammanfogas till endast en ram.
initial Sätter värdet på egenskapen till dess standardvärde.
inherit Ärver egenskapen från ett förälder-element.

Exempel

table {
    border-collapse: collapse;
}

Border Spacing

Border-spacing-egenskapen används för att sätta avståndet mellan ramar för närliggande celler. Kan bara användas om border-collapse: separate.

Värde Beskrivning
length Specifiserar ett avstånd för närliggande cellers ramar.
initial Sätter värdet på egenskapen till dess standardvärde.
inherit Ärver egenskapen från ett förälder-element.

Exempel

table {
    border-collapse: separate;
    border-spacing: 10px 50px;
}

Caption Side

Caption-side-egenskapen definierar vart placeringen av tabellbeskrivningen ska vara.

Värde Beskrivning
top Beskrivningen placeras ovanför tabellen. Används som standard.
bottom Beskrivningen placeras under tabellen.
initial Sätter värdet på egenskapen till dess standardvärde.
inherit Ärver egenskapen från ett förälder-element.

Exempel

caption {
    caption-side: top;
}

Empty Cells

Empty-cells-egenskapen specifiserar om tomma cellers ramar och bakgrunder ska visas eller inte. Kan bara användas om ramarna för tabellen är separata.

Värde Beskrivning
show Ramar och bakgrunder syns för tomma celler. Används som standard.
hide Inga ramar eller bakgrunder syns för tomma celler.
initial Sätter värdet på egenskapen till dess standardvärde.
inherit Ärver egenskapen från ett förälder-element.

Exempel

table {
    border-collapse: separate;
    empty-cells: hide;
}

Formulär

Här kommer även exempel på hur man kan designa formulär med hjälp av CSS.

Exempel

HTML

<form>
    <label>Användarnamn
        <input type="text" name="username">
    </label>
    <label>
        Lösenord
        <input type="password" name="password">
    </label>
    <input class="btn" type="submit" name="submit" value="Login">
</form>

CSS

form {
    border: 1px solid #c6c7cc;
    overflow: hidden;
    width: 240px;
    padding:10px;
}

label {
    color: #395870;
    display: block;
    font-weight: bold;
    margin-bottom: 20px;
}

input {
    background: #fff;
    border: 1px solid #c6c7cc;
    color: #636466;
    padding: 6px;
    margin-top: 6px;
    width: 100%;
}

.btn {
    background: #293f50;
    border: 0;
    color: #fff;
    padding: 8px 16px;
}
