Bilder och bakgrunder

Bakgrounds-egenskaper används för att definiera bakgrundseffekter för ett element.

Background-attachment

Bestämmer om bakgrunden följer med när man skrollar på hemsidan.

Kod

div {
 background-image: url('image.jpg')
 background-attachment: #b0c4de;
}

Background-color

Specifiserar en bakgrundsfärg för elementet.

Kod

div {
 height:200px;
 width:200px;
 background-color: #b0c4de;
}

Resultat

A div-element.

Background-image

Specifiserar en bekgrundsbild för elementet.

Kod

div {
 height:200px;
 width:200px;
 background-image: url('image.jpg');
}

Resultat

A div-element.

Background-position

Sätter startpunktspositionen för en bakgrundsbild.

Kod

div {
 height:200px;
 width:100%;
 background-repeat: no-repeat;
 background-image: url('image.jpg');
 background-position: center;
}

Resultat

A div-element.

Background-repeat

Definierar om bakgrundsbilden ska upprepas eller inte.

Kod

div {
 height:200px;
 width:100%;
 background-repeat: no-repeat;
 background-image: url('image.jpg');
}

Resultat

A div-element.

Bild och text

Om man vill ha bilder i en text så kan det vara användbart att använda sig av egenskapen 'float'. Float gör det möjligt att positionera en bild åt höger eller vänster i texten utan att texten avbryts, den flödar istället runt bilden.

Kod

img {
            float: right;
            margin: 10px;
            }
            .float h2 {
            clear:both;
            }
