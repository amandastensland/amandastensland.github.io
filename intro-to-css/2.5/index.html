Länktillstånd

Länkar kan bli stylade olika beroende på vilket länktillstånd som länken befinner sig i. Det finns fyra tillstånd som en länk kan befinna sig i.

  • a:link - en normal, obesökt länk
  • a:visited - en länk som användaren har besökt
  • a:hover - en länk när användaren har musen på den
  • a:active - en länk som användaren klickar

Cursor-egenskapen specifiserar muspekarens utseende vid pekande på ett element och kan till exempel användas i kombination med länkar för att förstärka länktillstånden.

Exempel

a:link {
color: black;
}

a:visited {
color: grey;
}

a:hover {
color: red;
cursor: crosshair;
}

a:active {
color: yellow;
}
