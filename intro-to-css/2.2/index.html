Text och listor

Fontfamiljer

  • Serif: Fonten har små linjer på bokstavskanterna.
  • Sans-Serif: Har inga linjer på bokstavskanterna.
  • Monospace: Alla monospace-bokstäver har samma breli.

Enheter

  • p { font-size: [value] }
  • xx-small: Väldigt liten text.
  • x-small: Ännu mindre text.
  • small: Liten text.
  • medium: Medium text.
  • large: Stor text.
  • x-large: Ännu större text.
  • xx-large: Väldigt stor text.
  • %: Textens storlek visad i %.
  • em: Textens storlek visad em.

Färg

  • En egen definierad färg.
  • p { font-color: red; }
  • Fördefinierad färg.
  • p { font-color: #B4D455; }

Teckensnitt

  • font-family: Font-family egenskapen anger teckensnittet för ett element.
  • p { font-family: "Arial", "Helvetica"; }
  • font-stretch: Med font-stretch egenskapen kan du göra texten bredare eller smalare.
  • p { font-stretch: expanded; }
  • font-size: Ändrar storleken på texten.
  • p { font-size: 3em; }
  • font-size-adjust: Webbläsaren justerar textstorleken så att den är samma oavsett vilken font som används.
  • p { font-size-adjust: 0.58; }
  • font-style: Stylar texten.
  • p { font-style: italic; }
  • font-variant: Konverterar lowercase till små uppercase-bokstäver.
  • p { font-variant: small-caps; }
  • font-weight: Definierar hur tjocka bokstäverna ska vara.
  • p { font-weight: bold; }

Text

  • letter-spacing: Sätter avståndet mellan bokstäverna.
  • p { letter-spacing: 2px; }
  • word-spacing: Sätter avståndet mellan ord.
  • p { word-spacing: 10px; }
  • line-height: Sätter avståndet till nästa textrad.
  • p { line-height: 10px; }
  • text-align: Sätter textens justering.
  • p { text-align: center; }
  • text-decoration: Sätter dekoration på texten.
  • p { text-decoration: underline; }
  • text-shadow: Lägger till en skugga bakom texten.
  • p { text-shadow: 2px 2px #B4D455; }
  • text-indent: Indenterar den första raden av texten.
  • p { text-indent: 50px; }
  • text-transform: Transformerar texten mellan stora och små bokstäver.
  • p { text-transform: uppercase; }
  • white-space: Specifiserar om texten ska brytas till nästa rad eller inte.
  • p { white-space: ; }

Stycken

  • p:first-line Markerar den första linjen av varje p-element.
  • p:first-line { text-transform: uppercase; }
  • p:first-letter Markerar den första bokstaven av varje p-element.
  • p:first-letter { text-transform: uppercase; }

Listor

  • list-style-image: Specifisera en bild istället för den vanliga listmarkeraren.
  • p { list-style-image: url('image.gif); }
  • list-style-position: Specifiserar om listmarkerarna ska visas inuti eller utanför textflödet.
  • p { list-style-position: "Arial", "Helvetica"; }
  • list-style-type: Sätter stilen på listmarkeraren.
  • p { list-style-type: upper-roman; }
