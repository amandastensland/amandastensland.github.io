HTML

    Kapitlet introducerar HTML5 som är den senaste standarden för att strukturera innehåll i webbläsare.
  • 1.2 Semantiska element för struktur
    Kapitlet introducerar till semantiska element för struktur som används för att på ett tydligt sätt dela upp ett HTML-dokument i olika delar.
  • 1.3 Semantiska element för text
    Kapitlet introducerar till semantiska element för text som används för att beskriva innehåll på ett meningsfullt sätt.
  • 1.4 Formulär
    Kapitlet visar hur man skapar och använder ett formulär.

    Kapitlet introducerar till progressivt nerladdat ljud och film för webben.
